# Deep Track — Deep Work Studio

Deep Track is a premium, high-performance productivity engine designed for elite focus. It transforms your workstation into a distraction-free sanctuary by combining high-precision timing, system-level notification suppression, and an immersive study environment.

## 📥 How to Install & Update

**This repository hosts the official releases and updates for Deep Track.** 
The application features a built-in auto-updater. Once installed, you will seamlessly receive new features and performance patches without needing to download new files manually.

### Initial Installation:
1. Navigate to the **[Releases](../../releases/latest)** section on the right side of this page.
2. Download the latest `Deep Track Setup.exe` file.
3. Run the installer to setup the application on your PC.

*(Note: During the first installation, Windows SmartScreen may prompt a warning since this is an indie developer certificate. Click "More Info" -> "Run Anyway" to proceed.)*

## 🚀 Key Features

* **Obsidian Flow UI**: A deep charcoal, electric purple, and teal aesthetic built with high-end glassmorphism to minimize visual noise.
* **Strict Protocol Lockdown**: Full-screen enforcement that disables the taskbar, blocks system distractions, and suppresses Windows notifications.
* **Instant Media Engine**: Native, zero-latency control for background media (Play/Pause/Skip) and an integrated system volume slider.
* **Analytic Studio**: Interactive Chart.js visualizations for your activity heatmaps, daily screen time, and focus streaks.
* **Immersive WebView**: Built-in, ad-free study browser that prevents accidental navigation and time-wasting domains.
* **Zen Journaling**: Persistent, built-in scratchpad for capturing thoughts without leaving your flow state.
