## ⚠️ Important Note: Windows Defender / SmartScreen warning

Because Deep Track is an open-source, indie application that is not signed with a paid commercial code signing certificate (which costs hundreds of dollars annually), **Windows Defender SmartScreen will show a warning dialog upon your first installation.** 

This is standard behavior for all unsigned desktop executables. The application is entirely safe, open-source, and does not collect any personal data.

### How to Bypass and Install:
1. Double-click the downloaded **`Deep Track Setup.exe`** file.
2. When the blue **"Windows protected your PC"** SmartScreen window appears, click the **`More info`** link under the text.
3. A button saying **`Run anyway`** will appear in the bottom right corner.
4. Click **`Run anyway`** to launch the installer and set up Deep Track on your computer.

## 🚀 Key Features

*   **Obsidian Flow UI:** A deep charcoal, electric purple, and teal aesthetic built with high-end glassmorphism to minimize visual noise.
*   **Draggable Floating Focus Bubble:** A sleek, movable timer overlay that floats on top of your work. Constraints are bound precisely to your physical monitor boundaries using pointer capture.
*   **Registry Auditory Blackout:** System-level notification suppression that temporarily silences Windows notification alerts and alarms during active focus sessions.
*   **Strict Protocol Lockdown:** Full-screen enforcement that disables the taskbar, blocks desktop access, and keeps you anchored to your tasks.
*   **Immersive WebView:** Built-in ad-free study browser that locks onto your study tools and blocks distracting domains.
*   **Instant Media Sync:** Reads active media metadata from Windows WinRT and provides native, zero-latency controls (Play/Pause/Skip) and system volume controls directly in the flow UI.
*   **Analytic Studio:** Interactive charts powered by Chart.js displaying focus intensity heatmaps, session distributions, and daily targets.
*   **Zen Journaling:** A persistent, local scratchpad for capturing thoughts and ideas without breaking focus.
*   **Startup Self-Healing:** Automatic detection of background directory modifications, ensuring clean startup registry operations.
